3、git config --list	查看
4、git init		在项目中初始化
5、git add 文件名	添加到暂存区域
6、git commit -m "说明，做了什么"
7、git status	查看当前状态
8、git reset HEAD	把添加到暂存区域的文件恢复回去
	git reset HEAD~10	把10次之前的快照放回到暂存区域
9、git checkout -- 文件名		暂存区域的文件覆盖回已经被修改的文件(危险)
